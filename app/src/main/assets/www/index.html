Frequência Cardiaca Máxima


A Frequência Cardíaca Máxima (FCM) é o número máximo de batimentos que seu coração é capaz de efetuar em um minuto. Além de testes laboratoriais para verificar a FCM de um indivíduo, há algumas fórmulas para estimá-la, a mais usada é 220 - idade do individuo para o homem, já para a mulher a formula muda, ficando 226 - idade

Testes Físicos

Testes Físicos são testes realizados para determinar o seu desempenho em aspectos Físicos

--- > Flexão

Consiste em fazer baixar o corpo de forma uniforme até que o peito fique a uma
mão travessa do solo, sem lhe tocar e de seguida regressar a posição inicial. Este exercício está
voltado para o desenvolvimento do músculos peitorais e do tríceps, com benefícios auxiliares
para músculos como deltoide, serrátil anterior e coracobraquial. Esse exercício é básico para
treinamento em academias e, especialmente, para treinamento militar.

  • Ler Mais sobre Flexão

--- > Abdominal

Abdominal supra é o exercício onde você deita no solo, flexiona (dobra) os joelhos coloca as mãos cruzadas na frente do tronco, após este procedimento flexiona-se à frente fazendo o peito se aproximar das coxas voltando a deitar-se novamente. As mãos poderão ser atrás da nuca com os dedos entrelaçados. O exercício abdominal é um dos mais conhecidos exercícios para desenvolvimento e fortalecimento da musculatura abdominal, principalmente do músculo reto abdominal.
